1. Accédez à votre dashboard Neon (console.neon.tech).
2. Sélectionnez votre Projet.
3. Cherchez la section "Connection Details".
4. Copiez la "Connection String" (elle commence par postgres://...).
   Ce sera votre variable DATABASE_URL.

---------------------------------------------------------

CONFIGURATIONS POUR RENDER (BACKEND) :

1. Créez un nouveau "Web Service".
2. Connectez votre GitHub repo.
3. Paramètres :
   - Name: mj-chauffage-backend
   - Root Directory: backend  <-- TRES IMPORTANT
   - Environment: Node
   - Build Command: npm install && npm run build
   - Start Command: npm run start:compiled

4. Environment Variables (Variables d'environnement) :
   - NODE_ENV = production
   - DATABASE_URL = (Collez le lien Neon copié à l'étape 1)
   - JWT_SECRET = (Inventez un mot de passe long et compliqué)
   - SESSION_SECRET = (Inventez un autre mot de passe long)

5. Cliquez sur "Create Web Service".
